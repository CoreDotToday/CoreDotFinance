    1-4. PER 데이터 불러오기¶

    PER 데이터 뿐만 아니라 EPS, PER, BPS, PBR, 주당배당금, 배당수익률 등의 데이터도 얻을 수 있다.

    In [4]:
from coredotfinance.data import KrxReader
krx = KrxReader()

krx.read('005930', kind='per', start='2021-06-16', end='2021-06-22')
