    삼성전자 평가¶

    coredotfinance에서 불러온 주식 데이터와 cufflinks를 이용해 삼성전자의 현재 주식을 평가해 본다 cufflinks를 사용해보면 coredotfinanace의 데이터와는 역방향의 데이터를 사용한다는 것을 알 수 있다. 따라서 cufflinks를 사용할 때는 데이터의 방향을 고려해주어야 한다.

    [26]:

    from coredotfinance.data import KrxReader

krx = KrxReader()
df = krx.read('005930', start='2020-01-01', end='2021-07-15')
qf=cf.QuantFig(df.iloc[::-1],title='삼성전자')
qf.add_bollinger_bands(60)

    카카오 평가¶

    coredotfinance의 종목검색 기능을 사용해서 종목코드를 얻고 이를 통해서 데이터를 불러온다.

    [17]:

    krx.search('카카오')

    [17]:

    ('카카오', 'KR7035720002', '035720')

    [11]:

    from coredotfinance.data import KrxReader

krx = KrxReader()
df = krx.read('035720', start='2020-01-01', end='2021-07-15', adjust=True)
df.drop(df.loc[(df.index > '2021-04-11') & (df.index < '2021-04-15')].index, inplace=True)
qf=cf.QuantFig(df.iloc[::-1],title='카카오')
qf.add_bollinger_bands(60)
fig = qf.iplot(asFigure=True)
fig.write_image('hi.png')
