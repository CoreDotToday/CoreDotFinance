coredotfinance.crypto.get_ohlcv(ticker: str = 'BTCUSDT', interval='1d', start=None, end=None, limit=None) → pandas.core.frame.DataFrame¶

Parameters

  • ticker (str, optional) -- Binance Ticker, by default "BTCUSDT"

  • interval (str, optional) -- 조회 간격 설정, by default "1d" (1m, 3m, 5m, 15m, 30m, 1h, 2h, 4h, 6h, 8h, 12h, 1d, 3d, 1w, 1M)

  • start (str, optional) -- 조회 시작 날짜(YYYYMMDD), by default 최근 날짜

  • end (str, optional) -- 조회 끝 날짜(YYYYMMDD), by default 최근 날짜

  • limit (int, optional) -- 조회 개수, by default 500

Returns

대상 Ticker의 조회 조건에 맞는 일시별 시가/고가/저가/종가/거래량 DataFrame
